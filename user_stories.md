# Casting Marketplace User Stories

This document outlines the user stories for the three primary user roles in the casting marketplace application: Actors/Talent, Casting Directors/Producers, and Agents. It also includes common stories applicable to all users.

---

## 1. Actor/Talent User Stories

**Goal:** To find and secure acting roles by showcasing skills and experience.

### Profile Management
- **As an Actor,** I want to create a detailed professional profile with my physical attributes (height, eye color, etc.), so that Casting Directors can determine if I fit a role's basic requirements.
- **As an Actor,** I want to upload multiple high-resolution headshots and production stills, so that I can display my various looks.
- **As an Actor,** I want to embed video reels from YouTube or Vimeo and upload video files directly, so that I can showcase my performance skills.
- **As an Actor,** I want to upload my resume in PDF format, so that Casting Directors can review my complete work history and training.
- **As an Actor,** I want to add and display my specific skills (e.g., "fluent in Spanish," "horseback riding," "stage combat"), so that I can be discovered in specialized searches.
- **As an Actor,** I want to manage my union status (e.g., Equity, SAG-AFTRA), so that I only apply for and am considered for appropriate jobs.

### Job Discovery and Application
- **As an Actor,** I want to search for casting calls on a central dashboard, so that I can see all available opportunities.
- **As an Actor,** I want to filter casting calls by location, role type (film, theatre, commercial), pay scale, and union status, so that I only see relevant opportunities.
- **As an Actor,** I want to save my filter settings, so that I don't have to re-enter my search criteria every time I visit.
- **As an Actor,** I want to view detailed information for each casting call, including role description, production details, and audition dates.
- **As an Actor,** I want to submit myself for a role with a single click, using my default profile information, so that the application process is fast and efficient.
- **As an Actor,** I want to optionally add a short cover note or a specific headshot/reel when I apply, so that I can tailor my submission to the role.
- **As an Actor,** I want to see the status of all my applications (Submitted, Viewed, Shortlisted, Cast), so that I can track my progress.

## 2. Casting Director/Producer User Stories

**Goal:** To find the most suitable talent for projects in an efficient and organized manner.

### Project & Role Management
- **As a Casting Director,** I want to create a new project and add multiple roles to it, so that I can manage my entire casting process in one place.
- **As a Casting Director,** I want to specify detailed requirements for each role, including age range, physical attributes, required skills, and compensation, so that I receive relevant applications.
- **As a Casting Director,** I want to save a role as a draft before publishing it, so that I can finalize all details before it goes live.
- **As a Casting Director,** I want to set an application deadline for a role, so that submissions are automatically closed at the specified time.

### Talent Discovery & Management
- **As a Casting Director,** I want to view all applicants for a specific role on a single, organized dashboard.
- **As a Casting Director,** I want to filter and sort applicants by their attributes, skills, and union status, so that I can quickly narrow down the candidates.
- **As a Casting Director,** I want to view an actor's full profile, including all photos, videos, and resumes, directly from the applicant dashboard.
- **As a Casting Director,** I want to change the status of an applicant (e.g., Shortlisted, Rejected, Callback), so that I can keep track of my casting workflow.
- **As a Casting Director,** I want to create and manage multiple shortlists of talent for different roles or projects.
- **As a Casting Director,** I want to proactively search the entire actor database with advanced filters, so that I can discover talent that has not applied for my role.

---

## 3. Agent User Stories

**Goal:** To manage a roster of talent and efficiently find them suitable job opportunities.

### Roster Management
- **As an Agent,** I want to build and manage a roster of actors by inviting them to be represented by me.
- **As an Agent,** I want to have a dashboard view of all my clients and their current application statuses, so that I have a clear overview of my business.
- **As an Agent,** I want to access and edit my clients' profiles (with their permission), so that I can ensure their materials are always up-to-date.

### Job Submission & Tracking
- **As an Agent,** I want to view all public casting calls, just as an actor would.
- **As an Agent,** I want to submit one or more of my clients for a role in a single action, so that I can work efficiently.
- **As an Agent,** I want to track the status of all submissions made on behalf of my clients, so that I can follow up with Casting Directors.
- **As an Agent,** I want to receive notifications intended for my clients (e.g., audition requests), so that I can coordinate their schedules.

---

## 4. Common User Stories (All Users)

### Authentication & Authorization
- **As a User,** I want to sign up for a new account using my email and a password.
- **As a User,** I want to log in to my account securely.
- **As a User,** I want to be able to reset my password if I forget it, so that I can regain access to my account.
- **As a User,** I want my session to remain active so I don't have to log in every time I visit the site.
- **As a User,** I want to be able to log out of my account.

### Communication & Notifications
- **As a User,** I want to receive in-app and email notifications when my application status changes or I receive a message.
- **As a User,** I want a private messaging system to communicate directly with other users (e.g., a Casting Director messaging a shortlisted Actor).
- **As a User,** I want to manage my notification preferences, so that I only receive updates that are important to me.

### General Usability
- **As a User,** I want the website to have a clean, intuitive, and responsive design that works on both desktop and mobile devices.
- **As a User,** I want to have a personal dashboard that shows me the most relevant information and actions as soon as I log in.
- **As a User,** I want to be able to easily edit my own account information (name, email, etc.).
